# Video-ASRforEFL
A collection of Video&amp;ASR code designed for EFL classes
VideoSpeakingPractice1.html, videoResponse1.js, and video_speaking.css are old files with little feedback, but that provide the base for the user to "talk" to a video.

# Files for Nakamura et al. (2026)
The html file and js file titled super_interactive show the code for the watcher as an interlocutor. A running version can be found here:
https://springsenglish.online/speaking/easyspeaking.html

The videos with the watcher as an observer can be found in the running version here:
https://springsenglish.online/flashcards/isfc.html
